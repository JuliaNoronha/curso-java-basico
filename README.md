# ☕ Curso de Java Básico - Loiane Groner

Este repositório contém as anotações, exemplos de aula e resoluções dos exercícios práticos desenvolvidos durante o **Curso de Java Básico**, ministrado pela [Loiane Groner](https://github.com/loiane).

---

## 📌 Sobre o Curso

O curso aborda a sintaxe e os conceitos fundamentais do ecossistema Java, partindo do zero até conceitos estruturais da linguagem.
